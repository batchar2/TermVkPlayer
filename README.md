# TermVkPlayer
Плеер для VK, с текстовым интерфейсом. Написан на питоне, с использованием ncurses и gstreamer.

Статус: в разработке, относительно юзабельно.

## Установка

sudo pip install vk_api

Устновите пакет gstreamer-0.10
